## Getting Started

First, install the packages:

```bash
.../backend$ yarn 
.../frontend$ yarn
# or with npm
npm install
```

Then run the development servers:


```bash
.../backend$ yarn develop
.../frontend$ yarn dev
# or with npm
npm run develop
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result of the frontend.
